Comments about bluemix:

I det stora hela är vi ganska nörjda med bluemix,
när väl allt har fungerat har det fungerat hyffsat smidigt.

Tyvärr har det varit en hel del nertid, minst två gånger
har vi haft avbrott i utveckarkonsollen på flera timmar.

Dessutom är utveckarkonsollen fruktansfärt långsam
vilken gör den rätt jobbig att använda. Men kanske inte ett
jättestort problem då tiden man spenderar där är hyffsat lite.

Vårt största problem var att sätta upp ett projekt med
JSF stöd för bluemix. Det fanns inget exempelprojekt
för JSF med bluemix så det tog oss en bra stund att få
till det med rätt bibliotek osv.

Kombinationen Wesphere för lokal testing och bluemix
som slutgiltlig platform funkade mycket bra. Vi konfigurerade
websphere till att ansluta till databasen som hostades på
bluemix.
